<h1>
<a name="leap-unity-websocketclient" class="anchor" href="#leap-unity-websocketclient"><span class="mini-icon mini-icon-link"></span></a>Leap Unity WebSocketClient</h1>

<p>WebSocket Client for using Leap with Unity standard version.</p>

<p>WebSocket part using code from : Nugget Web Socket Server
<a href="http://nugget.codeplex.com/">http://nugget.codeplex.com/</a></p>

<p>JSON parser use parser from : light-weight-json-library
<a href="http://code.google.com/p/light-weight-json-library/">http://code.google.com/p/light-weight-json-library/</a></p>

<h1></h1>

<h1>
<a name="notes" class="anchor" href="#notes"><span class="mini-icon mini-icon-link"></span></a>Notes</h1>

<ul>
<li>This WebSocket Client has been testing with Leap's WebSocket Server ONLY!</li>
<li>It seem that "id" number will jump from time to times (not a perfect sequence).</li>
<li>Currently this code just get JSON frame data like javascript version gets.
It leave parts that mapped this data into Leap's data, Feel free to contribute :).</li>
<li>To use in Unity : 

<ul>
<li>Add folder "WebSocketClient" and all files in this folder into your project.</li>
<li>You can use "WebSocketClientBehavior.cs" in "Unity" folder as a starting point.</li>
</ul>
</li>
</ul></article>
